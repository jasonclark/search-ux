Privacy - Ten Blue Links

With this project, I am interested in preserving the privacy of users while allowing for customizations that add value to searching and browsing. In the interest of transparency, I'm listing the sources of the data that allowed for the enhancements and facets for the application

  • Rudimentary sentiment analysis and semantic evaluation of the search query
  • Cues from the "machine": HTTP Headers, global server variables, and location info from the network or browser
  • IP and PII (personally identifiable information) anonymized search log and web analytics data

I'm interested in finding the line where utility and adding value with this data cross into notions of surveillance, If you have thoughts or want to learn more, contact Jason A. Clark or find me on twitter @jaclark.
